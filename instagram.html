Como publicar vídeos do perfil @inclusao.gf.oficial

Devido a limitações de acesso direto via API público, este espaço oferece duas opções:

  1. Colar URLs das publicações (ex.: https://www.instagram.com/p/XXXXXXXX/) no arquivo data/insta_posts.json — a página irá renderizar automaticamente os embeds.
  2. Usar o link do perfil para abrir no Instagram (botão abaixo).

Instruções rápidas: copie URLs dos posts que contenham vídeos e cole no arquivo mencionado; a página os exibirá.
