Projetos de Inclusão

Projeto 1 - Sala Multifuncional Comunitária

Criar uma sala multifuncional para AEE que atenda à comunidade escolar, com recursos digitais, materiais adaptados e formação contínua para mediadores.

Projeto 2 - Programa Família Ativa

Oficinas e grupos de apoio para famílias, promovendo troca de experiências, orientações sobre direitos e mediação para o acompanhamento escolar.

Projeto 3 - Tecnologia Assistiva na Escola

Implementação de tecnologias assistivas (softwares comunicativos, teclados adaptados, dispositivos de acessibilidade) com formação para uso em sala.

Projeto 4 - Inclusão Sensorial

Ambientes e rotinas sensoriais na educação infantil e séries iniciais para ampliar a participação de crianças com transtornos do processamento sensorial.

Projeto 5 - Capacitação e Emprego para Cuidadores

Formação profissional e intermediação para contratação de cuidadores escolares, com certificação e supervisão pedagógica.
